# Pizza Sales Performance Dashboard

## 📌 Project Overview
This project began as a guided learning exercise using a commercial pizza sales dataset to master Power BI fundamentals. To elevate the dashboard from a static report to an interactive tool, I independently designed and integrated custom slicers, allowing users to dynamically filter the data by pizza category and size.

## 🛠️ Tools & Skills Applied
* **Power BI:** Building dashboard pages, formatting visuals, and prioritizing user experience (UX).
* **Interactive Filtering:** Implementing targeted slicers to allow stakeholders to drill down into specific product performance.
* **DAX & Data Modeling:** Calculating foundational metrics like total revenue, total orders, and average order value.

## 📊 Key Business Metrics
The dashboard tracks standard commercial performance indicators, allowing users to dynamically explore:
* **Peak Operations:** Identifying the busiest days of the week and peak months for order volume.
* **Revenue Tracking:** Monitoring total revenue and average order values across different timeframes.
* **Product Performance:** Utilizing custom-built slicers to isolate sales data by specific pizza sizes and flavor categories to inform inventory decisions.
